# Ricardo's Portfolio

A modern, high-end portfolio built with React, Tailwind CSS v4, and Framer Motion.

## Tech Stack

- **Framework:** React 19 (Vite)
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Deployment:** GitHub Pages (via GitHub Actions)

## Getting Started

### Prerequisites

- Node.js 20+
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ricardoml96/ricml96.github.io.git
   cd ricml96.github.io
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## Deployment

The project is automatically deployed to GitHub Pages whenever you push to the `main` branch.

The deployment configuration is located in `.github/workflows/deploy.yml`.

## Customization

- **Content:** Update `src/App.jsx` to change your bio, projects, and social links.
- **Styling:** Modify `src/index.css` to update theme colors and global styles.
- **Images:** Add your assets to the `public/` or `src/assets/` directory.
